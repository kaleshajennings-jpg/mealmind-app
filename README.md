# 🍽️ MealMind — Family Meal Planner App

MealMind is a mobile-first meal planning app that helps families plan their week of meals, generate smart grocery lists, and watch how to cook every dish on YouTube.

---

## 🚀 Getting Started (Run Locally)

### Prerequisites
- [Node.js](https://nodejs.org/) (version 16 or higher)

### Install & Run

```bash
# 1. Clone or download this repo
git clone https://github.com/YOUR_USERNAME/mealmind-app.git

# 2. Go into the project folder
cd mealmind-app

# 3. Install dependencies
npm install

# 4. Start the app
npm start
```

The app will open at **http://localhost:3000** in your browser.

---

## 🌐 Deploy to Vercel (Free — Live Website in 2 Minutes)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"Add New Project"** → select `mealmind-app`
4. Click **Deploy** — done! ✅

You'll get a free live URL like: `mealmind-app.vercel.app`

## 📱 Features

- ✅ 5-step personalized onboarding
- ✅ 57+ meals across 8 cuisines (Soul Food, Caribbean, Mediterranean, Mexican, Italian, Asian, Indian, American)
- ✅ 7-day meal plan with breakfast, lunch & dinner
- ✅ YouTube "Watch How to Cook" button on every meal
- ✅ Smart grocery list with every ingredient per meal
- ✅ Family mode (1–6 servings)
- ✅ Meal swap & "Surprise Me" regeneration
- ✅ Save favorite meals
- ✅ Grocery list with checkboxes & delivery integration
- ✅ Works as a Progressive Web App (add to home screen)
